## 🎞️ About The Project

**Movie Alarm**은 인기 영화의 특별관(IMAX, Dolby Cinema, 4DX 등) 예매 오픈을 실시간으로 감지하여 디스코드 채널로 알림을 보내주는 서비스의 소개 페이지입니다.

치열한 예매 전쟁(용아맥, 코돌비 등)에서 승리하고 싶은 영화 팬들을 위해 제작되었습니다.

🔗 **Live Demo:** [https://jangsc0000.github.io/movie-alarm/](https://jangsc0000.github.io/movie-alarm/)

<br>

## ✨ Key Features

*   **⚡ Real-time Notification**: 예매 오픈 즉시 알림 발송
*   **🎥 Major Theaters**: CGV, Megabox 주요 지점 지원
*   **📱 Mobile Friendly**: 반응형 웹 디자인으로 모바일에서도 완벽한 가독성
*   **🎨 Dark & Cinematic UI**: 영화관의 몰입감을 살린 다크 테마 디자인
